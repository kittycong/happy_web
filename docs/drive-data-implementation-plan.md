# Drive 자료 반영 개선 계획

## 목표

Google Drive의 `가계부_2026.xlsx`를 기준으로 카드 혜택 계산기의 기본 입력값과 추천 설명을 개인 지출 구조에 맞춘다.

## 구현 범위

1. `data/household-2026-preset.json`을 읽어 기본 입력값을 채우는 프리셋 버튼을 추가한다.
2. 카드대금, 대출 이자, 기존 할부 상환은 혜택 계산용 소비에서 제외하고 별도 요약에 표시한다.
3. 보험료, 인터넷, 교통 충전, 공과금 같은 반복 결제처는 현재 카드와 추천 카드 배정을 비교한다.
4. 분석 기준을 `최근 안정 구간`과 `연간 평균`으로 나눠 선택할 수 있게 한다.
5. 추천 결과 카드에 유리한 이유와 실적 제외 리스크를 한 줄 설명으로 추가한다.

## UI 제안

- 지출 입력 상단에 `가계부 프리셋 적용` 보조 버튼 추가
- 최종 정리 탭에 `혜택 계산 제외 지출` 패널 추가
- 최종 정리 탭에 `자동납부 이동 후보` 패널 추가
- 추천 카드 카드 안에 `추천 이유` 문장 추가

## 계산 원칙

- 카드대금은 소비 카테고리가 아니므로 중복 계산하지 않는다.
- 대출/이체성 고정비는 카드 실적 및 할인 계산에서 제외한다.
- 할부 상환은 현금흐름에는 표시하되 신규 카드 추천 점수에는 기본 제외한다.
- 공과금은 계절 변동이 커서 최근 3개월 평균과 연간 평균을 모두 제공한다.
