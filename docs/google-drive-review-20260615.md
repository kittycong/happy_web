## 우선 적용해야 할 개선 방향

1. Drive/Sheets 또는 CSV에서 가계부 데이터를 불러와 월평균 지출 입력값을 자동 채운다.
2. 카드 추천 계산에서 카드대금, 대출 이자, 기존 할부 상환, 현금/이체 지출을 분리한다.
3. 보험료, 통신, 인터넷, 공과금, 교통 충전 같은 반복 자동납부 항목은 현재 카드와 추천 카드 배정을 비교한다.
4. 전체 평균, 최근 3개월 평균, 가장 최근 입력월 중 분석 기준을 선택할 수 있게 한다.
5. 추천 결과에 “왜 이 카드가 유리한지”를 지출 카테고리와 실적 제외 금액 중심으로 설명한다.
