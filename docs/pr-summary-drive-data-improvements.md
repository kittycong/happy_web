# PR 요약

## 변경 사항

- Google Drive 가계부 자료 기반 월 지출 프리셋 JSON 추가
- 카드 혜택 계산에서 제외해야 할 지출 유형 정의
- 반복 자동납부 항목과 현재 카드 배정 데이터 정리
- `index.html`에 적용할 구현 계획 문서화

## 기대 효과

- 사용자가 매번 월 지출을 손으로 입력하지 않아도 됨
- 카드대금/대출/할부 상환이 카드 혜택 계산을 왜곡하는 문제를 줄임
- 자동납부를 어떤 카드로 옮겨야 하는지 판단하기 쉬워짐
- 추천 결과가 실제 가계부 흐름에 더 가까워짐

## 후속 작업

- GitHub CLI 인증 후 대형 `index.html` 본문에 프리셋 적용 버튼과 로더 함수 반영
- GitHub Pages 배포 후 브라우저에서 프리셋 로드와 분석 플로우 검증
